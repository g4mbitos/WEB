Типи тегів Назва тегу Опис тегу
Для каркасу документа <html> Основний тег html-документу, в якому знаходяться всі інші теги
<head> Тег в якому задаються налаштування, містить інформацію для браузера
<body> Тег в якому міститься контент сторінки
<footer> Тег в якому міститься інформація про розробника і контактні дані
Для роботи з текстом <b> Робить текст жирним
<i> Робить текст курсивом
<u> Підкреслює текст
<br> Переводить текс на новий рядок
<p> Абзац
<h1> - <h6> Різні види заголовків
Для роботи з списками <ol> Нумерований список
<ul> Ненумерований список
<li> Рядок списку
Блочні елементи <div> Тег для роботи з блоками
<span> Тег для виділення інформації всередині інших тегів
Для роботи з таблицями <table> Контейнер для елементів таблиці
<th> Комірка заголовку таблиці
<tr> Рядок таблиці
<td> Комірка таблиці
Для роботи з посиланнями <a> Встановлює посилання або якор
